Gaming Behavior ML - Documentación

API para predecir el nivel de engagement de jugadores usando Machine Learning.

Endpoints

MétodoEndpointDescripción
GET/api/infoInformación de la API
GET/healthVerificar estado del servicio
GET/metricsMétricas de Prometheus
GET/docsSwagger UI interactivo
POST/predictPredecir engagement de un jugador
POST/predict/batchPredecir múltiples jugadores (máx 100)

Ejemplo de Predicción

POST /predict

{
  "Age": 22,
  "Gender": "Female",
  "Location": "USA",
  "GameGenre": "RPG",
  "PlayTimeHours": 14.5,
  "InGamePurchases": true,
  "GameDifficulty": "Hard",
  "SessionsPerWeek": 15,
  "AvgSessionDurationMinutes": 135,
  "PlayerLevel": 72,
  "AchievementsUnlocked": 38
}

Respuesta

{
  "prediction": "High",
  "probabilities": {
    "Low": 0.028,
    "Medium": 0.072,
    "High": 0.9
  }
}

Modelo

Algoritmo: Random Forest Classifier

Accuracy: ~90%

Clases: Low, Medium, High

Tecnologías

FastAPI, Scikit-learn, Pandas, Pydantic, SlowAPI, Prometheus
